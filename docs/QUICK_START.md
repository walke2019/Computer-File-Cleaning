# 快速开始指南

## 安装依赖

```bash
npm install
```

## 开发运行

```bash
# 启动应用
npm start

# 开发模式（带日志）
npm run dev
```

## 打包发布

### Windows

```bash
npm run build:win
```

生成文件位于 `dist/` 目录：
- `智能文件整理助手_1.0.0_win.exe` - 便携版

### macOS

```bash
npm run build:mac
```

生成文件位于 `dist/` 目录：
- `智能文件整理助手_1.0.0_mac.dmg` - 安装包
- 支持 Intel (x64) 和 Apple Silicon (arm64)

### 所有平台

```bash
npm run build:all
```

## 功能测试

### 1. 垃圾清理
1. 点击"垃圾清理"标签
2. 点击"开始扫描"
3. 查看扫描结果（按分组显示）
4. 选择要清理的类别
5. 点击"清理选中项"

### 2. 大文件管理
1. 点击"大文件管理"标签
2. 选择驱动器和最小文件大小
3. 点击"开始扫描"
4. 查看文件类型分布图表
5. 选择文件并删除或移至回收站

### 3. 磁盘信息
1. 点击"磁盘信息"标签
2. 查看各磁盘使用情况
3. 查看可视化存储占用

### 4. 系统信息
1. 点击"系统信息"标签
2. 查看操作系统、CPU、内存等信息

## 常见问题

### Q: 扫描速度慢？
A: 可以调整最小文件大小，或排除某些目录（在设置中添加白名单）

### Q: 无法访问某些目录？
A: Windows 需要管理员权限，macOS 需要在系统偏好设置中授权

### Q: 如何恢复误删的文件？
A: 使用"移至回收站"功能而不是直接删除，可以从回收站恢复

### Q: 支持哪些语言？
A: 目前仅支持简体中文

## 开发建议

### 代码规范
- 使用 `PlatformAdapter` 处理平台差异
- 使用 `path.join()` 构建路径
- 避免硬编码平台特定路径
- 添加平台检测条件

### 性能优化
- 使用增量扫描
- 限制并发文件操作
- 缓存扫描结果
- 使用 Web Workers（未来）

### 安全建议
- 始终提供预览功能
- 实现撤销机制
- 使用回收站而非永久删除
- 添加白名单保护重要目录
